# Git基础

## 基础知识：
### commit 说明
  * Header
    * Type：
      * fix   ----- 解决Bug
      * feat  ----- 新增功能
      * docs  ----- 文档更新（documention）
      * style ----- 代码格式更新（不影响代码运行）
      * reflactor - 重新构建
      * chore ----- 项目构建过程或者辅助工具的变动
      * test  ----- 增加测试模块

    * scope:
      * 更新影响范围

    * subject:
      * 更改的目的
